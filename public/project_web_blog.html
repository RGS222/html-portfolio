Web Development Project - Shout Web Blog

This capstone project is part of the "Complete Full-Stack Web Development Bootcamp" by Dr. Angela Yu at Udemy.

The goal of this project is to create a blog web application using Node.js, Express.js, and EJS. The application will allow users to create, view, update and delete blog posts. Posts will persist with the use of a data file. Styling will be an important aspect of this project to ensure a good user experience.
