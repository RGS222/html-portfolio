Game Design Automation System Project

The system is built to assist the mathematicians to enter game data, manipulate the data to proper format and create the game assets. It contains user interface applications, SQL Server database, back-end services and other worker services for processing the data. The system uses MVVM pattern, and Entity Framework, WCF and WPF technology. I am involved in the implementation of the user application, the control model and the database in the system.
