A Builder with Code

Game Design Automation System Project

The system is built to assist the mathematicians to enter game data, manipulate the data to proper format and create the game assets. It contains user interface applications, SQL Server database, back-end services and other worker services for processing the data. The system uses MVVM pattern, and Entity Framework, WCF and WPF technology. I am involved in the implementation of the user application, the control model and the database in the system.

Reel Analysis Tool Project

The tool is used to simulate the spinning of slot reels to provide statistical data for the game play. It contains a user application integrated with Excel and services running on servers processing large quanitity of data, supported by MS SQL Server database. It also comes with an android app for user to check the processing status.

Aries Tools Project

This is a collection of standalone Python automation applications with Windows context menu integration. These applications assist game math developers to perform time-consuming repetitive tasks.
